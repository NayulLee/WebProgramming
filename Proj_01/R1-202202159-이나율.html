🐰2. 텍스트 관련 태그

HTML 태그는 크게 블록(block)과 인라인(inline)으로 나뉜다.
이 차이는 화면에 배치되는 방식의 차이이다.

🥕 블록 태그

    특징
  • 항상 새 줄에서 시작
  • 한 줄 전체(가로)를 독점해서 차지한다. 따라서 옆에 다른 요소가 올 수 없다.

대표 태그
p 문단(Paragraph)
h1 ~ h6 제목(heading)을 표시
div 구역(divide)을 나누는 컨테이너
ul 순서 없는 리스트를 표시

🥕 인라인 태그

    특징
  • 블록 태그 안에 들어가서 일부러 표시된다.
  • 옆의 다른 인라인 요소랑 나란히 배치할 수 있다.

대표 태그
strong 굵게 표시(bold)
a ~ h6 하이퍼링크
img 이미지
span 구간 나누기
(짧은 텍스트 스타일 적용할 때 자주 사용)

👉 가장 많이 쓰는 건 span, 특정 글자에 색/스타일 주고 싶을 때 사용한다.

🐰3. 리스트 관련 태그

🥕 리스트 만들기 3가지 태그


  1. 순서 있는 리스트 ol...ol
    항목들에 자동으로 번호 1, 2, 3이 붙는다.
    안에 들어가는 항목은 li(list item)태그로 표현한다
  2. 순서 없는 리스트 ul...ul
    번호 대신 불릿으로 표시된다
    li(list item)태그로 항목을 작성한다.
  3. 정의 리스트(definition list) dl...dl
    용어와 그에 대한 설명을 표현할 때 사용한다.
    • dt: definition term(용어)
    • dd: definition description(설명)

🐰4. 테이블 관련 태그

🥕 표 만드는 데 사용되는 태그

  1. 표 전체를 담는 컨테이너<ol...ol>
    표의 시작과 끝을 감싸는 태그
    안에 행<tr>과 열<th>, <td>가 들어간다.
  2. 표 제목<caption>
    표 제목을 지정할 때 사용, 보통 표 맨 위에 나타난다.
  3. 표의 헤딩 그룹<thread>
    표의 머리 부분, 열 제목, th들을 묶어주는 역할
    여러 행이 필요할 때도 묶어서 관리 가능하다
  4. 표의 바닥 그룹<tfoot>
    표의 바닥 부분. 합계, 평군 등 마지막 줄에 들어갈 데이터를 묶을 때 사용
  5. 표의 본문 그룹<tbody>
    실제 데이터 셀들을 모아서 관리한다
    <tr>행 단위로 나누어 넣는다.
  6. 행(row)<tr>
    표의 한 줄을 의미, 각 행 안에 열 제목<th>또는 데이터 <td>넣는다.
  7. 열 제목 셀<tht>
    머리글 셀, 보통 글자가 굵게 표시되고 가운데 정렬된다.
    어떤 열의 의미인지 알려주는 셀
  8. 데이터 셀<td>
    실제 데이터를 넣는 셀, <td>와 달리 일반 글자 스타일
