🐰2. 텍스트 관련 태그

HTML 태그는 크게 블록(block)과 인라인(inline)으로 나뉜다.
이 차이는 화면에 배치되는 방식의 차이이다.

🥕 블록 태그

특징

  • 항상 새 줄에서 시작
  • 한 줄 전체(가로)를 독점해서 차지한다.
    따라서 옆에 다른 요소가 올 수 없다.
대표 태그
  • p : 문단(paragraph)
  • h1 ~ h6 : 제목(heading)
  • div : 구역 나누기(divide)
  • ul : 순서 없는 리스트

🥕 인라인 태그

특징
  • 블록 태그 안에 들어가서 일부러 표시된다.
  • 옆의 다른 인라인 요소 랑 나란히 배치할 수 있다.
대표 태그
  • strong : 굵게 표시
  • a : 하이퍼링크
  • img : 이미지
  • span : 구간 나누기(짧은 텍스트 스타일 적용할 때 자주 사용한다)
👉 가장 많이 쓰는 건 span특정 글자에 색/스타일 주고 싶을 때 사용한다.
